#Auto Accessories Website

A modern, fully responsive website built with HTML, CSS, and JavaScript to showcase auto accessories. The project uses CSS variables for easy theme customization and includes interactive and visually appealing design elements.

#Features

Responsive layout for desktop, tablet, and mobile devices.

CSS variables for centralized color and theme management.

Interactive navigation menu with hover effects.

Home, About, Services, Reviews, Contact, and Footer sections.

Animated services slider and hover effects on images and buttons.

Contact form designed for future backend integration.

Supports dark-theme styling via CSS variables.

#Technologies Used

HTML5 – Website structure.

CSS3 – Styling, variables, animations, and responsive design.

JavaScript – Interactive elements like sliders and nav toggle.

Google Fonts – Raleway, Barlow Condensed, Fjalla One, Itim, Oswald, PT Sans Narrow.

#How to Run

Clone or download the repository.

Open index.html in your browser.

Update CSS variables in style.css to customize colors and theme.

Add your own images in the assets/ folder.

#Future Improvements

Add a dark/light mode toggle using CSS variables and JavaScript.

Add more interactive sliders for products or services.

Improve accessibility and SEO for better usability.
